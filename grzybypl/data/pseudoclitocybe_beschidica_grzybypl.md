# Lejkownik beskidzki

## Uwagi

Gatunek odróżniający się od lejkownika kubkowatokapeluszowego ( Pseudoclitocybe cyathiformis ) bardziej obfitym, biało filcowatym owłosieniem powierzchni trzonu.
