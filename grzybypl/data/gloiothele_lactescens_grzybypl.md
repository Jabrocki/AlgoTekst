# Balonikowiec mleczny

## Występowanie, ekologia

W lasach liściastych, parkach, na drzewach przydrożnych, na pniakach i pniach żywych drzew i krzewów liściastych np. kasztanowiec krwisty ( Aesculus pavia ) , wierzba ( Salix ) , wiąz pospolity ( Ulmus minor ) i kalina ( Viburnum ) ; głównie lato-jesień.
