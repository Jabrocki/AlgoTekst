# Twardzioszek trzcinowy

## Występowanie, ekologia

Na pędach traw (często trzciny pospolitej ( Phragmites australis ) ), także na łodygach turzyca ( Carex ) i pokrzywy zwyczajnej ( Urtica dioica ) ; czerwiec-sierpień.
