# Pieniążek drobniutki

## Występowanie, ekologia

W lasach różnego typu, na starych owocnikach grzybów kapeluszowych ( pierścieniak ( Stropharia ) , mleczaj ( Lactarius ) , gołąbek ( Russula ) ); czerwiec-listopad.
