# Powleczniczka grzybolubna

## Występowanie, ekologia

Na apotecjach wełnokulka owcza ( Lasiosphaeria ovina ) rosnącej na opadłych gałęziach lipy drobnolistnej ( Tilia cordata ) ; owocniki pojawiają się w przeciągu sezonu wegetacyjnego, wyłącznie po kilku dniach chłodnych deszczów.
