# Grzybojadek łzawnikowy

## Występowanie, ekologia

Nie tworzy własnych owocników, rozwija się w plesze owocników innych gatunków z rzędu łzawnikowe ( Dacrymycetales ) np. łzawnik rozciekliwy ( Dacrymyces stillatus ) i innych z rodzaju łzawnik ( Dacrymyces ) .
