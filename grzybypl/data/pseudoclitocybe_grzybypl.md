# Lejkownik

## Owocniki

Owocniki podobne do tworzonych przez lejkówka ( Clitocybe ) ale odróżniające się następującymi cechami: - brak sprzążek, Kapelusz higrofaniczny, w ciemnych barwach, ciemnobrązowy, szarobrązowy.

## Uwagi

Z "lejkówkowatych" grzybów jeszcze dwa rodzaje mają amyloidalne zarodniki - jednogatunkowy rodzaj z pieprzniczka szarawa ( Cantharellula umbonata ) i pępkogrzybówka ( Pseudoomphalina ) . Od lejkownik ( Pseudoclitocybe ) różnią się one obecnością sprzążek.
