# Przylepnik łysawy

## Cechy charakterystyczne

Standardowe reakcje porostowe : kora: (wszystkie reakcje negatywne) K-, C-, KC-, P-, N-; miąższ: C+ czerwony (krótkotrwale! przy "drapaniu" miąższu), K- lub w miejscach pomarańczowo zabarwionych ( skyrina ) ±purpurowy; KC+ czerwony; P-, UV-.
