# Próchnilec nitkowaty (Xylaria filiformis)

## Występowanie

Na różnych szczątkach roślinnych - rozkładających się łodygach bylin, ogonkach liściowych paproci, itp.

## Uwagi

Podobne owocniki ma próchnilec owocolubny ( Xylaria carpophila ) / - wyrasta na owocni (osłonce) orzeszków buka.
