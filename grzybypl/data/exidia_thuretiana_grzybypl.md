# Kisielnica biaława

## Występowanie

Na martwym drewnie i korze różnych drzew i krzewów liściastych.

Czerwona Lista [125 ] — R–rzadki - potencjalnie zagrożony wymarciem

## Uwagi

Na drewnie iglastym wyrasta podobny trzęsak mózgowaty ( Naematelia encephala ) / - pasożytuje on na grzybni skórnika krwawiącego ( Stereum sanguinolentum ) /
