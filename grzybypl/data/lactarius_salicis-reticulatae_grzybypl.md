# Mleczaj wierzby żyłkowanej

## Występowanie, ekologia

W piętrze alpejskim, w żyznych zbiorowiskach z dębik ośmiopłatkowy ( Dryas octopetala ) i wierzba żyłkowana ( Salix reticulata ) ; na ziemi.
