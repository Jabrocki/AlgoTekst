# Grzybolubka lepka

## Występowanie

W lasach z grabami i bukami, na martwych owocnikach grzybów z rodzaju mleczaj ( Lactarius ) , gołąbek smaczny ( Russula delica ) i gołąbek śmierdzący ( Russula foetens ) ; sierpień-listopad.

Czerwona Lista [125 ] — V–narażony na wymarcie

## Asterophora lycoperdoides (bull.) ditmar grzybolubka purchawkowata

Także na martwych owocnikach gołąbków tworzy nieco większe owocniki grzybolubka purchawkowata ( Asterophora lycoperdoides ) .
