# Woskobłonka białożółtawa

## Występowanie, ekologia

W lasach liściastych, na pniakach, leżących pniach i gałęziach drzew liściastych, np. olszy szarej ( Alnus incana ) i innych olsza ( Alnus ) , Betula ; kwiecień-październik.
