# Szyszkówka gorzkawa (Strobilurus tenacellus)

## Uwagi

Owocniki są makroskopowo identyczne z innymi szyszkówkami - zobacz opis szyszkówka świerkowa ( Strobilurus esculentus ) . Szyszkówka tęporozwierkowa ( Strobilurus stephanocystis ) ma cystydy szeroko-maczugowate).

## Cechy mikroskopowe

Wysyp zarodników jasno-kremowy. **Zarodniki** cylindryczno-elipsoidalne, gładkie, hialinowe, nieamyloidalne, 5-7.5 × 2.5-4 µm.

## Występowanie, ekologia

Gatunek pospolity. W lasach z udziałem sosny, na opadłych szyszkach sosny zwyczajnej ( Pinus sylvestris ) i sosny czarnej ( Pinus nigra ) ; wczesną wiosną, od kwietnia, rzadko później.

## Jadalne czy trujące? 🟡🟢

Jadalna o ile nie jest istotnie gorzka w smaku. [134 ]

## Komentarz

W. Wójciak: szyszkówka gorzkawa ( Strobilurus tenacellus ) i szyszkówka tęporozwierkowa ( Strobilurus stephanocystis ) z moich fotografii występują obok siebie w lesie sosnowym z domieszką liściastych. Owocniki szyszkówki tęporozwierkowej ( Strobilurus stephanocystis ) mają kapelusz jaśniejszy, blaszki białe z odcieniem żółtawym (woskowym).
