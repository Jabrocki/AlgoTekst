# Czyreń

## Opis

W Polsce występuje kilkanaście gatunków. Owocniki są wieloletnie, różnych kształtów, twarde, sucho-korkowate lub zdrewniałe. Miąższ zwarty, przeważnie ciemno zabarwiony. Hymenofor rurkowy, zwykle wielowarstwowy, przegrody między rurkami są dość szerokie, pory dość drobne.
